# How to Insert Images in the Syncfusion Angular RTE Using a File Manager

**Repository Description**  
This repository contains a **quick‑start Angular sample** that demonstrates how to **browse and insert images** into the Syncfusion **Angular Rich Text Editor (RTE)** using a **File Manager**.

The sample illustrates integrating a file‑browsing experience with the Rich Text Editor, allowing users to select images from a file system and insert them directly into editor content.

## Project Overview
The purpose of this project is to help developers understand how to configure and use the [Syncfusion Angular Rich Text Editor](https://www.syncfusion.com/angular-components/angular-rich-text-editor) with a File Manager for image insertion. It provides a reference implementation for adding file‑browser support to a rich text editing workflow in an Angular application.

## Features
- Integration of **Syncfusion Angular Rich Text Editor**
- Browse image files using a **File Manager**
- Insert selected images into the editor content
- Angular‑based quick start project
- Live reload during development

## Prerequisites
Ensure the following requirements are met before running this project:
- Angular **4+**
- TypeScript **2.6+**
- Node.js and npm
- A supported modern web browser

## Installation

### Clone the Repository
Clone the repository and navigate to the project directory:
```bash
git clone https://github.com/SyncfusionExamples/how-to-insert-images-in-the-angular-rte-using-a-file-manager.git
```
### Install Dependencies
Install the required Angular and Syncfusion packages:
```bash
npm install
```
### Running the Application
Start the Angular development server using:
```bash
ng serve
```
Navigate to the following URL in your browser:
```
http://localhost:4200/
```
The application automatically reloads when you modify source files.

## Usage
Once the application is running:
- Open the Syncfusion Angular Rich Text Editor.
- Use the file browser integration to browse image files.
- Select an image to insert it directly into the editor content.
This setup provides a seamless image‑insertion experience within the Angular RTE.

## Configuration
The file browser configuration is handled through the Rich Text Editor’s file browser settings. You can customize:
- File source paths
- Accepted image formats
- File Manager behavior for browsing and selection
